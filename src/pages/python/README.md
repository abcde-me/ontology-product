# Notebook 页面组件结构

## 概述

这是一个基于 Arco Design 和 React 构建的 Notebook 页面，提供了类似 Jupyter Notebook 的交互式数据分析和处理环境。

## 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│                     Notebook 页面                            │
├─────────────────┬───────────────────────────────────────────┤
│   左侧 Tab 区域   │              主内容区域                   │
│                 │                                           │
│ ┌─────────────┐ │  ┌─────────────────────────────────────┐  │
│ │    文件      │ │  │           标签页头部                  │  │
│ ├─────────────┤ │  ├─────────────────────────────────────┤  │
│ │  开发工具    │ │  │           工具栏                      │  │
│ ├─────────────┤ │  ├─────────────────────────────────────┤  │
│ │   数据源     │ │  │          工作区内容                   │  │
│ └─────────────┘ │  └─────────────────────────────────────┘  │
└─────────────────┴───────────────────────────────────────────┘
```

## 组件结构

```
src/pages/notebook/
├── index.tsx                 # 主页面组件
├── index.scss               # 主页面样式
├── components/              # 子组件目录
│   ├── index.ts            # 组件导出文件
│   ├── NotebookTabContent.tsx  # Tab 内容组件
│   ├── NotebookMainContent.tsx # 主内容区域组件
│   ├── NotebookWorkspace.tsx # 工作区组件
│   ├── NotebookToolbar.tsx # 工具栏组件
│   ├── NotebookContent.tsx # 内容区域组件
│   ├── CodeEditor.tsx      # 代码编辑器组件
│   └── *.scss              # 各组件样式文件
└── README.md               # 说明文档
```

## 功能特性

### 1. 左侧 Tab 区域

- **文件 Tab**: 管理 Notebook 文件，支持搜索和新建
- **开发工具 Tab**: 算子库和工具管理
- **数据源 Tab**: 数据库和文件数据源管理
- **树形目录**: 每个 Tab 都有对应的树形结构展示

### 2. 主内容区域

- **标签页头部**: 支持多个 Notebook 标签页，动态增减
- **工具栏**: 运行、导出、调用算子等操作按钮
- **工作区**: 代码编辑和内容展示区域

### 3. 标签页功能

- **动态标签页**: 支持创建、关闭、切换多个 Notebook
- **标题编辑**: 双击或点击编辑按钮可修改标签页标题
- **智能关闭**: 关闭标签页时自动切换到相邻标签页
- **最小保护**: 至少保留一个标签页，避免无内容状态

### 4. 核心功能

- **文件管理**: 支持创建、搜索、切换笔记本文件
- **代码编辑**: 支持 Python、Markdown、SQL 等语言
- **单元格管理**: 可以添加、删除、编辑代码和文档单元格
- **工具集成**: 算子库和数据处理工具
- **数据源管理**: 数据库连接和文件导入

### 5. 交互体验

- **响应式设计**: 适配不同屏幕尺寸
- **现代化 UI**: 基于 Arco Design 的设计系统
- **流畅动画**: 平滑的过渡效果
- **Tab 切换**: 垂直 Tab 布局，清晰的功能分区
- **悬停效果**: 标签页编辑按钮的优雅显示

## 技术栈

- **React 18**: 前端框架
- **TypeScript**: 类型安全
- **Arco Design**: UI 组件库
- **SCSS**: 样式预处理器

## 使用说明

1. **切换功能区域**: 点击左侧的"文件"、"开发工具"、"数据源" Tab
2. **管理文件**: 在"文件" Tab 中创建、搜索、选择笔记本
3. **使用工具**: 在"开发工具" Tab 中选择算子库中的工具
4. **连接数据**: 在"数据源" Tab 中配置数据库和文件连接
5. **管理标签页**:
   - 点击"+"按钮创建新标签页
   - 点击"×"按钮关闭标签页
   - 点击编辑按钮或双击标题编辑标签页名称
6. **编辑代码**: 在主工作区添加和编辑代码单元格
7. **运行代码**: 点击工具栏的"运行"按钮执行代码

## 布局特点

### 左侧 Tab 结构

- **垂直 Tab 导航**: 清晰的功能分区
- **搜索和新建**: 每个 Tab 都有对应的搜索框和新建按钮
- **树形目录**: 层级化的内容展示
- **图标支持**: 文件和文件夹图标区分

### 主内容区域

- **三层布局**: 标签页头部 → 工具栏 → 工作区
- **动态标签页**: 支持多个 Notebook 同时打开
- **智能标题编辑**: 内联编辑，支持回车确认和失焦保存
- **功能工具栏**: 常用操作按钮
- **灵活工作区**: 可扩展的内容编辑区域

### 标签页特性

- **卡片式设计**: 现代化的标签页外观
- **悬停编辑**: 鼠标悬停时显示编辑按钮
- **智能切换**: 关闭当前标签页时自动切换到相邻标签页
- **最小保护**: 确保始终至少有一个标签页存在

## 开发规范

- 使用 TypeScript 进行类型检查
- 遵循 React Hooks 最佳实践
- 组件采用函数式编程风格
- 样式使用 SCSS 模块化组织
- 保持代码整洁和可维护性
- 组件职责单一，便于测试和维护
- 状态管理清晰，避免复杂的状态逻辑
