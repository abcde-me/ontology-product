# modaforge-front

ModaForge 多模态数据治理平台前端框架

## 项目功能说明

### 数据集详情页面 (DatasetDetail)

数据集详情页面是多模态数据治理平台的核心功能模块，用于展示和管理数据集的详细信息。

#### 功能描述

**基本信息展示**
- 数据集名称、标签、创建人、生成模型等基础信息
- 当前版本号显示，带有"最新版本"标识
- 创建时间、更新时间、描述说明等详细信息
- 支持编辑功能，可修改数据集基本信息

**数据内容管理**
- ID精确查询：通过ID号快速定位特定数据条目
- 关键字搜索：支持在介绍、内容、回复字段中进行模糊搜索
- 数据列表展示：包含ID、介绍、内容、回复、状态等字段
- 操作功能：编辑单条数据、继续生成内容
- 分页展示：支持自定义每页显示数量和页面跳转

**版本历史管理**
- 版本列表展示：显示所有历史版本及描述信息
- 版本回滚：支持回滚到指定历史版本
- 更改追踪：显示每个版本的更改数量
- 版本详情：查看特定版本的详细信息

#### 使用方法

1. **访问页面**：通过路由 `/tenant/compute/modaforge/datasetManagement/detail/:id` 访问
2. **编辑信息**：点击"编辑"按钮可修改基本信息
3. **搜索数据**：选择搜索类型后输入关键词进行搜索
4. **管理内容**：在数据内容标签页中查看和操作具体数据
5. **版本管理**：在版本历史标签页中管理版本信息

#### 参数说明

- `id`: 数据集唯一标识符，通过URL参数传递
- `searchType`: 搜索类型，支持 'id' | 'keyword'
- `searchValue`: 搜索关键词字符串
- `currentPage`: 当前页码，默认为1
- `pageSize`: 每页显示数量，默认为10

#### 返回值说明

组件返回一个完整的数据集详情界面，包含：
- 面包屑导航组件
- 基本信息卡片组件
- 数据管理标签页组件
- 编辑弹窗组件
