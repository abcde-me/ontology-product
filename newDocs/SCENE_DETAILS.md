# RAG 分段列表 - 5种场景详解

## 场景概览

| 场景 | 名称 | 特点 | 使用场景 |
|------|------|------|---------|
| 1 | 基础文本 | 简单的文本分段 | 普通文档、合同等 |
| 2 | 分层级分段 | 目录树 + 分层级列表 | 结构化文档、法律文件 |
| 3 | 图文混合 | 文本 + 图片 + 弹窗 | 报告、演示文稿 |
| 4 | PPT展示 | 幻灯片 + 分段列表 | 演示文件、培训资料 |
| 5 | 表格分段 | 表格查看器 + 分段表格 | 数据表、统计报表 |

---

## 场景1：基础文本分段

### 功能描述
最基础的分段场景，左侧显示PDF原文件，右侧显示分段列表。

### 页面布局
```
┌─────────────────────────────────────────┐
│ Header (文件路径和名称)                  │
├──────────────────┬──────────────────────┤
│                  │ 分段列表头部          │
│   PDF查看器      ├──────────────────────┤
│   (50%)          │ 分段卡片1            │
│                  ├──────────────────────┤
│                  │ 分段卡片2            │
│                  ├──────────────────────┤
│                  │ 分段卡片3            │
│                  │ (50%)                │
└──────────────────┴──────────────────────┘
```

### 交互功能
- ✅ 点击分段卡片选中
- ✅ Hover显示操作按钮
- ✅ 编辑分段内容
- ✅ 隐藏/显示PDF
- ✅ 自动保存

### 访问URL
```
http://localhost:3000/tenant/compute/modaforge/ragDetail?ragId=text-scene
```

### 关键组件
- `MainContent.tsx` - 主容器
- `SegmentList.tsx` - 分段列表
- `SegmentCard.tsx` - 分段卡片
- `SegmentCardContent.tsx` - 内容编辑

---

## 场景2：分层级分段 + 目录树

### 功能描述
支持最多5层的分层级结构，左侧PDF，中间目录树，右侧分层级分段列表。

### 页面布局
```
┌─────────────────────────────────────────────────────┐
│ Header (文件路径和名称)                              │
├──────────────────┬──────────────┬──────────────────┤
│                  │ 目录树       │ 分段列表头部      │
│   PDF查看器      │              ├──────────────────┤
│   (33%)          │ 第一章       │ 分段卡片1        │
│                  │ ├ 第一节     │ (缩进20px)       │
│                  │ ├ 第二节     ├──────────────────┤
│                  │ 第二章       │ 分段卡片2        │
│                  │ ├ 第一节     │ (缩进40px)       │
│                  │ (33%)        │ (33%)            │
└──────────────────┴──────────────┴──────────────────┘
```

### 特殊功能
- 目录树最多5层（超过5层打平为第5层）
- 点击目录树节点，右侧分段列表自动滚动到对应分段
- 点击分段卡片，目录树对应节点高亮
- 分段卡片根据level显示不同的缩进

### 访问URL
```
http://localhost:3000/tenant/compute/modaforge/ragDetail?ragId=hierarchical-scene
```

### 关键组件
- `HierarchicalSceneContent.tsx` - 场景容器
- `DirectoryTree.tsx` - 目录树
- `HierarchicalSegmentList.tsx` - 分层级列表
- `HierarchicalSegment` - 类型定义

### Mock数据结构
```typescript
{
  sceneType: 'hierarchical',
  segments: [
    {
      id: 'seg_h_001',
      content: '...',
      level: 2,
      parentId: 'dir_1_1',
      ...
    }
  ],
  directory: [
    {
      id: 'dir_1',
      label: '第一章',
      level: 1,
      children: [
        {
          id: 'dir_1_1',
          label: '第一节',
          level: 2,
          segmentIds: ['seg_h_001']
        }
      ]
    }
  ]
}
```

## 场景3：图文混合分段

### 功能描述
分段中包含文本和图片，点击图片可弹窗放大查看。

### 页面布局
```
┌─────────────────────────────────────────┐
│ Header                                   │
├──────────────────┬──────────────────────┤
│                  │ 分段列表头部          │
│   PDF查看器      ├──────────────────────┤
│   (50%)          │ 分段卡片1            │
│                  │ ┌──────────────────┐ │
│                  │ │ 文本内容         │ │
│                  │ ├──────────────────┤ │
│                  │ │ 图片1 (可点击)   │ │
│                  │ │ 图片2 (可点击)   │ │
│                  │ └──────────────────┘ │
│                  │ (50%)                │
└──────────────────┴──────────────────────┘
```

### 特殊功能
- 分段卡片显示文本和图片
- 点击图片弹窗放大（最大高度800px）
- 弹窗宽度自适应
- 点击X或弹窗外区域关闭

### 访问URL
```
http://localhost:3000/tenant/compute/modaforge/ragDetail?ragId=image-text-scene
```

### 关键组件
- `ImageTextSceneContent.tsx` - 场景容器
- `ImageTextSegmentList.tsx` - 图文列表
- `ImageTextSegmentCard.tsx` - 图文卡片
- `ImageModal.tsx` - 图片弹窗

### Mock数据结构

## 场景4：PPT展示

### 功能描述
左侧显示PPT幻灯片，右侧显示分段列表。

### 页面布局
```
┌─────────────────────────────────────────┐
│ Header                                   │
├──────────────────┬──────────────────────┤
│ 幻灯片标题       │ 分段列表头部          │
│ ┌──────────────┐ ├──────────────────────┤
│ │              │ │ 幻灯片卡片1          │
│ │ 幻灯片内容   │ │ 第1页 / 财务成果    │
│ │              │ ├──────────────────────┤
│ │              │ │ 幻灯片卡片2          │
│ └──────────────┘ │ 第2页 / 项目成果    │
│ ← 上一页 1/4 下一页 → │ (50%)                │
│ (50%)            │                      │
└──────────────────┴──────────────────────┘
```

### 特殊功能
- 左侧PPT查看器支持上一页/下一页
- 显示当前页码
- 右侧分段卡片显示幻灯片号和标题
- 点击分段卡片切换PPT页面

### 访问URL
```
http://localhost:3000/tenant/compute/modaforge/ragDetail?ragId=ppt-scene
```

### 关键组件
- `PptSceneContent.tsx` - 场景容器
- `PptViewer.tsx` - PPT查看器
- `PptSegmentList.tsx` - 分段列表
- `PptSegmentCard.tsx` - 分段卡片

### Mock数据结构

## 场景5：表格分段

### 功能描述
左侧显示表格查看器，右侧显示分段表格列表。

### 页面布局
```
┌─────────────────────────────────────────┐
│ Header                                   │
├──────────────────┬──────────────────────┤
│ 表格标题         │ 分段列表头部          │
│ ┌──────────────┐ ├──────────────────────┤
│ │ 列1 │ 列2 │ │ │ 表格卡片1            │
│ ├─────┼─────┤ │ │ 员工信息表 (4行)    │
│ │ 数据 │ 数据 │ │ ├──────────────────────┤
│ │ 数据 │ 数据 │ │ │ 表格卡片2            │
│ └──────────────┘ │ │ 销售数据表 (3行)    │
│ ← 上一个 1/2 下一个 → │ (50%)                │
│ (50%)            │                      │
└──────────────────┴──────────────────────┘
```

### 特殊功能
- 左侧表格查看器支持上一个/下一个
- 表格支持水平滚动
- 右侧分段卡片显示表格预览（前3行）
- 显示总行数

### 访问URL
```
http://localhost:3000/tenant/compute/modaforge/ragDetail?ragId=table-scene
```

### 关键组件
- `TableSceneContent.tsx` - 场景容器
- `TableViewer.tsx` - 表格查看器
- `TableSegmentList.tsx` - 分段列表
- `TableSegmentCard.tsx` - 分段卡片

## 通用交互功能

### 所有场景都支持
- ✅ 分段选中（边框和背景高亮）
- ✅ Hover显示操作按钮
- ✅ 编辑分段内容
- ✅ 点击外部自动保存
- ✅ 显示字符数和分段序号

### 操作按钮
- **编辑分段** (✎) - 进入编辑模式
- **分段详情** (ⓘ) - 查看详情（待实现）
- **溯源日志** (⟲) - 查看日志（待实现）

## 样式规范

### 颜色
- 高亮蓝色: `#007DFA`
- 边框灰色: `#e5e7eb`
- 背景浅蓝: `#eff6ff`

### 尺寸
- 分段卡片最大高度: 500px
- 图片弹窗最大高度: 800px
- 目录树最多5层

---

## 总结

5种场景覆盖了不同类型的知识库文档，通过统一的架构和接口，可以灵活地支持各种场景。每种场景都有其特定的交互方式和展示逻辑，但都遵循相同的设计原则和代码结构。
